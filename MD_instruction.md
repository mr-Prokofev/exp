# Инструкция для работы с Markdown

## Выделение текста
Добавляю всякую шелуху. К примеру для выделения текста **жирным**, нужно обрамить его звездочками (**).
Чтобы выделеть текст *курсивом*, нужно обрамить его звездочками (*).
